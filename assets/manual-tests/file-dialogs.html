🧪 Test des dialogues de fichiers

Page HTML simple pour tester les sélecteurs de fichiers et l'emplacement de sauvegarde dans Dumber.

Ouvrir cette page dans Dumber :
ENV=dev DUMBER_LOG_LEVEL=debug ./dist/dumber browse /home/brice/dev/projects/dumber/assets/manual-tests/file-dialogs.html

Note : le choix de dossier via webkitdirectory et la boîte de dialogue de sauvegarde via showSaveFilePicker() dépendent du moteur navigateur embarqué.

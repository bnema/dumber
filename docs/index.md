# Dumber Documentation

Dumber is a keyboard-driven browser for tiling window managers, built with GTK4 and WebKitGTK.

## What is Dumber?

A browser designed for Wayland compositors that works like your favorite terminal multiplexer:

- **Tiling Panes** - Split, stack, and organize pages like Zellij/tmux
- **Modal Keybindings** - Keyboard-driven workflow with mode indicators
- **Session Management** - Auto-save and restore workspaces
- **Built-in Ad Blocking** - UBlock-based network + cosmetic filtering
- **Launcher Integration** - Rofi/fuzzel with favicons and history
- **GPU Rendering** - Automatic VA-API detection for video playback

## Requirements

- Linux with Wayland (Sway, Hyprland, River, Niri, etc.)
- GTK4 and WebKitGTK 2.42+
- GStreamer (for media playback)
